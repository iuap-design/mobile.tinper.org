# 活动指示器 ActivityIndicator 

活动指示器。 表明某个任务正在进行中。


## 如何使用

```
import { ActivityIndicator } from 'tinper-libraui';

```

## 代码演示


## API


### ActivityIndicator

| 属性 | 说明 | 类型 | 默认值 |
|----|-----|------|------|
|animating |	显隐状态	|boolean	|true|
|size |	spinner大小，可选small/large	|string	|small|
|toast |	loading样式类型	|boolean	|false|
|text |	loading文本	|string|	-|
